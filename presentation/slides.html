Existing Solutions & Literature

Standard LLMs

Method:

Direct prompting (ChatGPT/Claude) for script generation.

Limitations:
  • Hallucination: No integrated fact-checking.
  • Single-turn: Lacks iterative refinement.
  • Text-only: Cannot produce actual video files.

Template Video Tools

Method (e.g., Canva/Vyond):

Manual drag-and-drop on pre-made timelines.

Limitations:
  • Human-Intensive: Still requires hours of manual labor.
  • Static (Cost): Hard to update content dynamically.
  • No Logic: System doesn't "understand" the content.
Our Novel Approach: Agentic Orchestration Combining LLM reasoning with code execution & RAG for autonomous production.
Autonomous Verifiable End-to-End

Deep Dive: Phase 2

The Agent Squad

Key "System Messages" that define the personality and responsibility of each agent.

Course Planner

Role: Architect

"You are an expert curriculum designer. Given a topic, you must output a structured JSON of modules, lessons, and learning outcomes. Ensure logical flow."

Researcher

Role: Fact Checker

"You have access to the Qdrant Vector Store. Validate every claim the writer makes. Provide academic citations and source URLs for all content."

Script Writer

Role: Storyteller

"Create engaging, conversational video scripts. Use helpful analogies. Mark visual cues in brackets [Show Diagram] for the Video Producer."

The Critic

Role: Quality Control

"Review the script for length and tone. If it's too boring or too long, reject it and ask the Writer to revise. Ensure strict formatting."

Video Producer

Role: Engineer

"Convert the final script into a MoviePy-compatible JSON. Select stock footage, define timings, and synchronization points for the TTS audio."
