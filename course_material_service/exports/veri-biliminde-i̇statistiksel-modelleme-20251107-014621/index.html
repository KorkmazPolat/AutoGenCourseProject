Knowledge Check (Quiz)

  1. 1. In a multiple regression output, the coefficient for an independent variable is reported as 2.5. What does this coefficient indicate?

    Rationale: The coefficient in a regression model indicates the expected change in the dependent variable for each one-unit change in the independent variable, holding all other variables constant.

  2. 2. Which of the following metrics is best used to assess the goodness-of-fit of a linear regression model?

    Rationale: R-squared is the most commonly used measure to evaluate how well the independent variables explain the variability of the dependent variable in a linear regression model.

  3. 3. When validating a model, why is cross-validation preferred over a simple train-test split?

    Rationale: Cross-validation mitigates the risk of overfitting and provides a more reliable estimate of model performance by averaging results over different subsets of data, thereby reducing variability.

  4. 4. If the p-value for a coefficient in a model output is reported as 0.03, what does this imply?

    Rationale: A p-value of 0.03 indicates that the coefficient is statistically significant, meaning the independent variable likely has a meaningful relationship with the dependent variable at the 0.05 level.

  5. 5. When interpreting the coefficients in a logistic regression model, which factor must be considered?

    Rationale: In logistic regression, coefficients describe the change in the log odds of the dependent variable; when exponentiated, they represent odds ratios, which quantify the relationship between the predictor and the outcome.
